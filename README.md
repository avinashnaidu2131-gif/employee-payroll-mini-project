# 💼 Employee Payroll Management System (Java)

## 📌 Project Overview

The Employee Payroll Management System is a desktop application developed using Core Java and Swing GUI.  
The system allows users to calculate employee salaries based on employee type (Manager or Developer).

This project demonstrates object-oriented programming concepts, exception handling, and GUI event handling as part of a B.Tech 2nd Semester mini project.

---

## 🚀 Features

- Add Employee ID and Name
- Enter Basic Salary
- Select Employee Type (Manager / Developer)
- Bonus calculation for Manager
- Overtime calculation for Developer
- Custom Exception for invalid salary
- Input validation with error handling
- Professional GUI layout with:
  - Title Panel
  - BorderLayout & JPanel
  - Colored Background
  - Styled Buttons
- Clear and Exit options
